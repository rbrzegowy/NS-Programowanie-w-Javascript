JS i DOM

1

Pobieranie elementów

  • getElementById(id) – jeden element po atrybucie id
  • querySelector(css) – pierwszy pasujący, selektory jak w CSS
  • querySelectorAll(css) – statyczna NodeList wszystkich pasujących
  • getElementsByClassName – żywa HTMLCollection (aktualizuje się przy zmianie DOM)
const btn = document.getElementById('ok')
const box = document.querySelector('.container')
const imgs = document.querySelectorAll('.thumb')
const prices = document.getElementsByClassName('price')

2

Zmiana zawartości elementu

  • innerHTML – nadpisuje zawartość jako HTML, uwaga na XSS!
  • textContent – bezpieczny text-only, tagi nie są interpretowane jako html
  • innerText – jak textContent, ale uwzględnia widoczność tekstu definiowaną w CSS
el.innerHTML = '<h1>Nowa treść</h1>'
el.textContent = 'Zwykły tekst'

3

Tworzenie nowego elementu

  • document.createElement(tag) – tworzy węzeł poza drzewem DOM
  • Nowy element jest "w powietrzu" – nie pojawi się na stronie bez inseracji
  • Można mu nadać atrybuty, klasy i treść przed dodaniem do DOM
const btn = document.createElement('button')
btn.textContent = 'Nowy przycisk'
btn.className = 'my-btn'

Tu pojawi się nowo utworzony element

4

Klonowanie elementu

  • element.cloneNode(true) – głęboka kopia całego poddrzewa (dzieci też)
  • cloneNode(false) – tylko element bez dzieci
  • Klon nie ma przypisanych event listenerów dodanych przez JS (addEventListener)
const klon = okButton.cloneNode(true)
klon.id = 'ok-klon'
container.appendChild(klon)
5

Modyfikacja dzieci elementu

  • parent.appendChild(el) – dodaje na końcu listy dzieci
  • parent.removeChild(el) – usuwa wskazane dziecko
  • parent.replaceChild(new, old) – zamienia jedno dziecko na inne
  • Jeśli element jest już w DOM – appendChild go przenosi, nie duplikuje
container.appendChild(okButton)
container.removeChild(cancelButton)
container.replaceChild(newEl, oldEl)

6

Wstawianie przed/po elemencie

  • Metody: el.before(), el.after()
  • Jeśli wstawiany element już jest w DOM – zostanie przeniesiony (nie zduplikowany)
  • Starsza alternatywa: parent.insertBefore(el, ref) – wstawia el tuż przed ref
container.insertBefore(newEl, imgList[0])

7

Style i klasy CSS

  • el.style.property – styl inline (camelCase, np. marginTop)
  • window.getComputedStyle(el) – live obiekt ze wszystkimi obliczonymi wartościami
  • el.className – pełne nadpisanie listy klas
  • el.style-box.add / remove / toggle / contains – precyzyjna praca z klasami
el.style.background = '#000'
getComputedStyle(el).marginTop
el.classList.toggle('active')

Styl i klasy zmieniane przez JS

8

Atrybuty i dataset

  • el.setAttribute(name, val) – ustawia atrybut HTML
  • el.removeAttribute(name) – usuwa atrybut
  • el.getAttribute(name) – odczytuje wartość atrybutu
  • el.dataset – obiekt z atrybutami data-* jako właściwościami
btn.setAttribute('disabled', 'true')
btn.removeAttribute('disabled')
div.dataset.id // <div data-id="p1">
thumb 1 (data-id="p1")

9

Rozmiary elementów i viewport

  • window.innerWidth / innerHeight – czysty viewport (bez pasków przeglądarki)
  • offsetWidth / offsetHeight – z paddingiem i borderem
  • clientWidth / clientHeight – bez bordera
  • scrollWidth / scrollHeight – pełna zawartość razem z ukrytą za scrollem
el.offsetWidth / el.clientWidth / el.scrollWidth
window.innerWidth / window.innerHeight
