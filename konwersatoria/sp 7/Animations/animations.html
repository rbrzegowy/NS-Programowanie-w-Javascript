Animacje i multimedia

1

Magic Camera (Canvas + kamera)

  • Pobranie obrazu z kamery przez getUserMedia.
  • Filtrowanie pikseli i render na canvas.
  • Zrzut zdjęcia przyciskiem Capture photo.
navigator.mediaDevices.getUserMedia(...)
ctx.getImageData(...) / ctx.putImageData(...)
2

Canvas slider

  • Ładowanie obrazów do tablicy Image.
  • Animacja przejścia między slajdami przez requestAnimationFrame.
  • Pętla normalna i pętla wstecz.
ctx.drawImage(image, x, y, w, h)
requestAnimationFrame(anim)

Image source: designbolts.com

3

SVG animation

  • Tworzenie elementów SVG przez createElementNS.
  • Animacja krzywych i kolorów.
  • Render kilku linii na jednym obszarze SVG.
document.createElementNS('http://www.w3.org/2000/svg', 'path')
4

Web Animation API

  • Animacja elementu przez element.animate(keyframes, options).
  • Kontrola stanu przyciskiem pauza/wznów.
  • Nasłuch zdarzenia zakończenia animacji.
const animation = box.animate([...], {...})
